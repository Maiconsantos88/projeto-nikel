# 💰 Projeto Nikel

Aplicação web de controle financeiro desenvolvida como prática de desenvolvimento front-end durante a formação em Desenvolvimento Web pela Growdev (Codaí 2.0).

---

## 🌐 Acesse o projeto

👉 https://maiconsantos88.github.io/projeto-nikel/

---

## 📱 Sobre o projeto

O Nikel é uma aplicação web que permite ao usuário gerenciar suas finanças de forma simples e intuitiva, controlando entradas, saídas e saldo.

Este projeto foi desenvolvido com foco em prática de desenvolvimento front-end, organização de código e construção de interfaces responsivas.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap

---

## ⚙️ Funcionalidades

* 🔐 Login de usuário
* 📝 Cadastro de conta
* 💸 Registro de entradas e saídas
* 📊 Visualização de saldo
* 📱 Layout responsivo (adaptado para mobile)

---

## 🎓 Formação

Projeto desenvolvido durante o curso de Desenvolvimento Web da Growdev (Codaí 2.0), onde foram aplicados na prática:

* Estruturação de páginas com HTML
* Estilização com CSS
* Interatividade com JavaScript
* Responsividade com Bootstrap
* Versionamento com Git e GitHub
